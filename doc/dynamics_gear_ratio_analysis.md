## 1. 连杆受力（矢状面静力 Newton-Euler 递推）

站立位姿（body_height=0.30 m，足端在髋正下方）解得 `q_hip=26.3°, q_knee=−56.3°`，
膝在 (0.080, −0.161)。从足端地反力 **F** 出发逐段做 ΣF=0 / Σm=0：

```
小腿段：R_knee = −(F + m_shank·g)              τ_knee = −[ (r_foot−knee)×F + (r_com−knee)×m_shank·g ]
大腿段：R_hip  = −(−R_knee + m_thigh·g)         τ_hip  = −[ (r_knee−hip)×(−R_knee) + (r_com−hip)×m_thigh·g ]
```

| 工况 | τ_hip | τ_knee | 膝反力 \|R\| | 髋反力 \|R\| |
|---|---|---|---|---|
| 静立 4 腿支撑（Fz=W/4=28.7 N） | −1.86 | +2.18 | 25.9 N | 20.8 N |
| trot 2 腿支撑（Fz=W/2=57.4 N） | −4.15 | +4.47 | 54.6 N | 49.4 N |
| 动态蹬地（Fz=1.5·W/2, Fx=0.4·Fz） | −12.00 | +1.98 | 90.1 N | 85.4 N |
| 深蹲（body_height=0.256，eval 实测低姿） | −13.87 | +5.49 | — | — |

**关节峰值力矩需求 τ_req ≈ 12 N·m**（动态蹬地的髋力矩主导；深蹲再加约 15%）。
连杆自重仅占地反力 3–5%，故关节力矩由**地反力 × 力臂**主导；站姿越低（深蹲）力臂越大，
力矩需求越高——这正是 eval 里「蹲低→小腿近满载」的力学根因。

## 4. 电机 T-N 曲线与减速比寻优

线性 T-N：`τ_rotor(ω) = 3.744·(1 − ω/189.9)`。反射到关节：
`τ_joint = N·η·τ_rotor`，`ω_joint = ω_rotor / N`。

**两条约束**（峰值力矩与峰值转速是两个不同时发生的工作点）：

```
力矩（支撑/蹬地，低速）：N·η·τ_rotor_peak ≥ τ_req   ⇒  N ≥ 3.56
转速（摆动，低力矩）：    ω_rotor_0 / N    ≥ ω_req   ⇒  N ≤ 10.66
```

**可行带 N ∈ [3.56, 10.66]**。余量平衡最优（令力矩余量 = 转速余量，
即 `N* = √(N_min·N_max)`）⇒ **N\* = 6.16**。

现用 N=6.33 的余量：

| | 可用 | 需求 | 余量 |
|---|---|---|---|
| 关节峰值力矩 | 21.3 N·m | 12 N·m | **1.78×** |
| 关节空载转速 | 30 rad/s | 17.8 rad/s | **1.68×** |

见 `outputs/dynamics/tn_envelope.png`（两个工作点都落在 N=6.33 包络下）、
`outputs/dynamics/gear_margin.png`（两条余量曲线在 N≈6.3 处交叉）。

## 5. 结论

1. **现减速比 6.33 已接近最优**：余量平衡最优 N\*=6.16 与之几乎重合；力矩余量(1.78×)
   与转速余量(1.68×)基本均衡。在 T-N 图上，支撑点和摆动点都落在 6.33 包络线之下。

2. **若要微调**：在可行带内
   - 主打**中低速 / 抗扰 / 负载**：上调到 **N≈8**，换 ~26% 力矩余量（峰值速度降到 24 rad/s，
     仍够 2× 当前步速）；
   - 主打**高速奔跑**：保持 6.33 或更低（更大速度余量）。

3. **真机「力矩/电流不足」的根因不是减速比**：电机输出峰值 23.7 N·m（关节 21.3 N·m）
   对 12 N·m 需求有 1.78× 余量。问题在于：
   - 仿真 `effort_limit_sim=12 N·m` **恰好卡在需求线、零余量** → 建议放到 **16–18 N·m**
     （仍 ≤ 电机峰值），给抗扰/深蹲留裕量；
   - 真机侧需核查**驱动器电流上限**与**母线电压在大负载下是否掉压限流**。

> 这回答了 todo.md 的关键分叉问题——「是电机/减速比选型不够，还是驱动器电流上限/供电不足」：
> **是后者**。减速比无需更换。
